# Changelog

## 1.2.1 — 2026-07-17

- 修复 Composer 运行中仍显示发送纸飞机的问题；现在会显示清晰的停止方块或暂停双竖线
- 任务完成后按钮会自动恢复为发送箭头，按钮颜色和交互态也随状态同步
- 内置 Dream Skin 引擎更新到 `1.4.1`；仍只使用本机回环 CDP，不修改官方应用

## 1.2.0 — 2026-07-17

- 新增主题内透明动画 Miku Future Pet，替代带系统阴影的原生浮动 Pet。
- 项目对话页支持 Pet 拖动、位置记忆、挥手和运行/等待/审阅状态动画；New task、设置页和
  展开的辅助模块会自动隐藏，避免遮挡真实 Codex 内容。
- 修复主题自身 DOM 监听造成的窗口闪烁与界面卡顿；流式消息不再反复触发整页重绘。
- 内置 Dream Skin 引擎更新到 `1.4.0`，Pet 图集通过 `hatch-pet` v2 透明校验。
- 仍只使用本机回环 CDP，不修改官方应用。
